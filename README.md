Description : REST API endpoint for translating English text to French using the POST method.

Live Demo Link : https://merely-testing.onrender.com/api/v1/translate
Tech Stack used : NodeJs, ExpressJs

Setup :
To set up the API locally, follow these steps:
1. Clone this repository to your local machine.
2. Install the required dependencies using pnpm install.
3. Start the server by running pnpm start.
4. The API will be accessible at http://localhost:PORT, where PORT is the port number specified in the configuration.
